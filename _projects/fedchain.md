Researched and implemented a proof-of-stake blockchain for federated learning, addressing performance and security flaws found in previous literature.

- Implemented the peer-to-peer network using bidirectional gRPC.
- Engineered a proof-of-stake consensus mechanism resilient to malicious model updates.
- Secured message integrity with asymmetric cryptography.
- Trained models on the blockchain using PyTorch and demonstrated improved security over vanilla federated learning.
